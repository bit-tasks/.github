### Bit Docker Image
You can use the [Bit Docker image](https://github.com/bit-tasks/bit-docker-image) with various CI/CD tools. Select from these available images:

- **Latest Stable:** 
  ```
  bitsrc/stable:latest
  ```
  
- **Nightly:** 
  ```bash
  bitsrc/nightly:latest
  ```
